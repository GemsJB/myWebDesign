W3C Web Standards
HTML5 and CSS3 specifications from the World Wide Web Consortium underpin Modules 2 and 3. Semantic markup, validation, and standards-compliant coding are assessed throughout Weeks 3–7.
Mozilla Foundation / MDN
MDN Web Docs serve as the primary reference for HTML, CSS, and JavaScript content across all modules. Learners are required to consult and cite MDN documentation in assignments.
WCAG 2.1 Accessibility Guidelines
Web Content Accessibility Guidelines are embedded in Module 2 (semantic HTML) and Module 6 (UI/UX and accessibility). Capstone projects are evaluated against WCAG AA criteria.
Responsive Web Design Standards
Mobile-first design, CSS Flexbox, Grid, and media queries (Module 3, Weeks 5–7) and Bootstrap's responsive grid system (Module 5, Week 10) align with industry responsive design standards.
Agile Collaborative Development
Agile practices — iterative delivery, peer review, pair programming, and version-controlled collaboration — are embedded in Module 5 (Git/GitHub, Week 11) and the Week 12 capstone.
Bloom's Revised Taxonomy
Cognitive levels escalate progressively: Remember/Understand (Wks 1–2) → Apply (Wks 3–4, 8–9) → Analyze (Wks 5–7) → Evaluate (Wk 11) → Create (Wk 12).
ABET Student Outcomes (CS)
SO1 (design/development), SO2 (teamwork), SO4 (professional ethics), and SO6 (communication) are addressed through labs, pair programming, Git collaboration, and capstone presentation.
UNESCO ICT Competency Framework
Digital literacy, creative design, and collaborative creation pillars are addressed throughout: computational thinking (Wks 8–9), creative interfaces (Wks 5–7), and teamwork (Wks 11–12).
